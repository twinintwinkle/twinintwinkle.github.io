To provide a smooth and good experience to user, also, to make the documentation more clear and easy to do debugging, I separate the program to three different stage section with block.

Stage 0 : Start the Program - Preference Numbers Input

Stage 1 : Store the Preference Number (if any) - Set Number Input

Stage 2 : Generate Random Number - Print Report to txt file

If there are any input error occurred, the program will return to the beginning of that stage and allow user to re-input their data, so they don’t have to restart the whole program if they just make a mistake of their input.
